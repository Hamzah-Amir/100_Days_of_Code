# Operators in Python

This directory demonstrates the different types of operators available in Python.

## Key Concepts

### Arithmetic Operators
- `+` : Addition
- `-` : Subtraction
- `*` : Multiplication
- `/` : Division
- `//` : Floor Division
- `%` : Modulus
- `**` : Exponentiation

### Comparison Operators
- `>` : Greater than
- `<` : Less than
- `==` : Equal to
- `!=` : Not equal to
- `>=` : Greater than or equal to
- `<=` : Less than or equal to

### Logical Operators
- `and` : Logical AND
- `or` : Logical OR
- `not` : Logical NOT
